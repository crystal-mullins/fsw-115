In this class I learned many things that made me a much better programmer.

Forms are really important because most websites require user input. Creating a great user experience often depends on how user friendly your application is. Forms make taking user input and adding an alert box easy.

Text boxs that take user input and store it in a database is crucial for most marketing applications. Learing how to create an alert box was also very useful. An alert box can be used to verify the user input. This can catch typos that can cause problems later.
